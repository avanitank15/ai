An intelligent chatbot designed for e-commerce platforms that prioritizes sustainability. It helps users find eco-friendly products, tracks orders, and toggles between "Standard" and "Sustainability" modes.

## Features
* **Sustainability Mode:** Filters product search results to show only eco-friendly items when enabled.
* **Order Tracking:** Users can track status using Order IDs (e.g., ORD-999).
* **Product Recommendations:** Keyword-based search for catalog items.

## Setup
1.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
2.  Run the application:
    ```bash
    python app.py
    ```
3.  Open your browser to `http://127.0.0.1:5000`.

## Demo Commands
* "Turn on eco mode"
* "Show me a toothbrush" (Notice how results change based on mode)
* "Track order ORD-999"
